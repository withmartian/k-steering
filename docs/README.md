# K-Steering Documentation

Welcome to the official documentation for **K-Steering** — a lightweight and flexible toolkit for steering large language model outputs with minimal overhead and maximum control.

---

## 📌 What is K-Steering?

K-Steering enables controlled generation from language models by:

- Applying steering vectors at inference time
- Supporting parameter sweeps to identify optimal steering strengths
- Integrating seamlessly with Hugging Face and local datasets
- Providing evaluation utilities for systematic benchmarking
- Offering structured configuration for reproducible experiments

Whether you're experimenting with interpretability, alignment, or controllable generation, K-Steering is designed to keep the workflow simple and modular.
